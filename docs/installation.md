# Installation

Compatibility with core CKAN versions:

| CKAN version | Compatible? |
|--------------|-------------|
| 2.9          | no          |
| 2.10         | yes         |
| 2.11         | yes         |
| master       | yes         |

Install the extension:
```sh
pip install ckanext-collection
```

Add `collection` to the `ckan.plugins` setting in your CKAN config file .
